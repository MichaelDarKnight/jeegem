朴槿惠四面楚歌 下一步何去何从？-搜狐新闻

　　自10月24日“亲信干政门”事件引爆以来，韩国社会反对朴槿惠政府的行动不断升级，参加者逐渐扩大到社会各阶层。12日更是爆发了2000年以来韩国规模最大的集会。13日，韩国检方还表示将对朴槿惠进行“面对面”调查。有分析认为，在民意倒逼下，朴槿惠政府或将尽快做出重大决断，以稳定政局。

　　百万人集会呼吁接受民意

　　12日，首尔市中心光化门广场一带挤满了集会的民众，要求朴槿惠下台呼声从下午持续到深夜。尽管警方估计参与民众约为26万人，但根据首尔市交通等部门有关统计数据，当天集会者多达百万人，且参与者包括各年龄、性别、地域、信仰和阶层的人群，代表性非常广泛。《京乡新闻》发表社论称，百万民众烛光集会，是要求朴槿惠下台的“命令”。

11月12日晚，人们在韩国首尔参加集会。 （新华社/路透）

　　13日，在野党阵营又发动了新一轮对朴槿惠政府的攻势。第一大在野党共同民主党再次敦促朴槿惠尽快自行下台。第二大在野党国民之党和第三大在野党正义党则表示，朴槿惠如再不主动宣布“退居二线”，将发起对她的弹劾。

　　在汹涌的民意面前，13日青瓦台召开记者会回应称，朴槿惠总统深感当前形势的严峻性，将负起责任，努力使国政恢复正常运转。青瓦台有关人士还透露，正在就所有各类善后方案进行探讨。但显然，区区只言片语已不可能让民众和舆论满意。

　　调查矛头直指总统本人

　　更令朴槿惠头疼的是，随着韩国检方对亲信干政门事件调查的不断深入，有越来越多的证据将嫌疑指向了她本人。

　　据韩国媒体报道，负责调查“亲信干政门”事件的韩国检方特别调查本部表示，从12日起，已陆续传唤去年7月曾与朴槿惠单独面谈的多家大企业负责人，包括三星和现代汽车等韩国主要财阀。检方称，将于15日或16日对朴槿惠进行面对面调查。

　　11月4日，在位于韩国首尔的总统府青瓦台，韩国总统朴槿惠发表对国民讲话。 新华社发（青瓦台总统府提供）

　　据悉，这将是有史以来韩国在任总统首次接受检方调查。尽管根据韩国法律规定，总统在任期间拥有不受刑事指控的特权，但此前朴槿惠在第二次对国民谈话中表示，愿接受检方调查。

　　韩国媒体分析称，朴槿惠接受调查的内容将包括是否存在滥用职权、人事舞弊、泄漏国家机密行为等。如被证实，虽然最终被起诉的可能性较小，但势必将对朴槿惠的威信造成进一步的打击。

　　朴槿惠或将做出重大决断

　　据盖洛普韩国11日公布的民调结果，进入11月以来，朴槿惠的支持率连续两周仅为5%左右，创历届韩国总统支持率的最低纪录，尤其是在19岁至29岁人群中的支持率几乎为零。韩国民众对朴槿惠政府的信任已所剩无几。

　　就连执政党内要求她下台的声音也不断增多。据韩国媒体报道，42名执政党议员13日提出，执政党应自行解体，朴槿惠应立即放弃一切权力。执政党前党首金武星更是公开表示，鉴于朴槿惠涉嫌公权私用，违反宪法规定，应对朴槿惠进行弹劾。有韩国舆论认为，目前韩国国会共计300个议员席位中，在野党及无党派议员共计171席，假如有29名执政党议员同意弹劾朴槿惠，就足以达到弹劾案通过所须的三分之二票数。

　　11月4日，在韩国首尔火车站，人们观看总统朴槿惠对国民讲话的电视直播。 （新华社记者姚琪琳摄）

　　韩联社13日发表评论称，到了该朴槿惠做出“重大决断”的时候了，她应当放弃一切权力，“站到国民与历史的面前”。韩国舆论认为，在空前压力面前，朴槿惠或提出新的善后方案，其中可能包括退出执政党、“退居二线”，乃至直接宣布下台。
