陕西干部上班看购物网站 质问记者乱拍啥(图)-搜狐新闻

作者：管理员 发布于：2016-08-18 10:44:43
该名工作人员打开的电脑页面并非工作页面，而是一服装购物网站。 华商报 图

　　上班期间浏览购物网站，面对拍照取证不仅没有悔意，还厉声质问：“你刚乱拍啥？”8月11日上午，华商报记者在西乡县市场监管局采访时，遇到如此尴尬一幕。

　　吹着空调撩起上衣 电脑显示在看购物网站

　　11日上午11时35分，华商报记者在西乡县市场监管局采访，进入三楼市场规范管理股时，办公室内空调大开，有一名男性工作人员。只见这名工作人员穿着短袖撩开上衣，露出肚皮，正专心致志地看着电脑。

　　随后，华商报记者走向前出示记者证并询问采访事宜，就在此时，记者看到，该名工作人员打开的电脑页面并非工作页面，而是一服装购物网站。

　　起初，该名工作人员并未关闭网页，发觉记者拍照取证时，才赶紧将购物网页关闭。随后，该名工作人员厉声质问：“你刚乱拍啥？”

　　华商报记者并未理会，而是希望能先完成采访，这名工作人员在跟局领导打电话请示后表示，“必须先通过办公室安排后，才能接受采访。”随后，记者解释，此前已前往办公室，是办公室工作人员让到市场规范管理股采访时，该名男子仍强烈要求记者先去办公室。

　　据工牌显示，这名工作人员名叫杨某，是该科室普通干部。

　　县纪委已核实 对违纪行为给予行政警告

　　随后，华商报记者与杨某来到二楼的综合办公室，一名女性工作人员说：“你稍等，我和领导汇报一下。”这名工作人员和杨某一起离开办公室后，记者一直在等待中。然而，上午11时55分，办公室女性工作人员却跟记者说，“领导不在，我们已经下班了，你下午3点再过来吧。”

　　说完，综合办公室所有工作人员全都离开，而此时距离12时下班还有5分钟。

　　随后，华商报记者将这一情况向西乡县纪委进行了反映，纪委工作人员表示，他们将立即展开调查。15日，华商报记者从西乡县纪委了解到，经过他们调查核实，已对此事件做出了处理决定。

　　经过调查核实，纪委等部门研究决定，对西乡县市场监管局干部杨某违纪问题给予行政警告处分。

　　此外，西乡县要求全县干部要举一反三，结合“两学一做”学习教育，时刻紧绷纪律之弦，不断强化作风纪律约束，真正把纪律和规矩立起来、严起来。
