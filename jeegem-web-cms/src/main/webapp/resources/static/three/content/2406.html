　　　　公司一妹纸刚拿到驾驶证就开车来上班，路上遇到正要打车的我，愣是把我别到绿化带里，我急了问：想什么呢？妹纸尴尬的说：我只是想和你打个招呼，结果忘了是开车出来的...

 

　　绿化带里爬出来终于打上了车，路上，遇到两个女的打架，都动上手了。

　　司机师傅突然问我：“你赶时间吗？”

　　我说：“不呀，怎么了？”

　　然后司机师傅靠边停车，指着打架的地方：“不着急，我们看会再走，我觉得快要扒衣服了！”

 

　　为什么打群架的时候，普遍会脱掉上衣？

　　假如脱掉裤子的话，总感觉哪里不对，气氛怪怪的。

 

　　　　重磅消息！麻瓜们请注意，罗琳阿姨这次真的又写哈利波特系列的故事了！

 

　　罗琳阿姨还是原来的罗琳阿姨，但是哈利波特已经不是原来的哈利波特了！

 

　　美国、日本、中国的青春片各有什么特点？

　　美国-主角参加各种社团、比赛，然后顺便谈个恋爱，最后既实现了个人价值，又抱得美人归，最后主角说，我们的青春是如此精彩！”―一段青春期奋斗史；

 

　　日本―校园热暴力、冷暴力，主角一开始被欺凌，然后在朋友、家人和梦中情人的激励下奋起，突然从小孩变成大人，靠自己的努力赢得大家和梦中情人的认可，最后主角说：“我们的青春值得奋斗！”―一段青春期的蜕变史；

 

　　中国―各种恋爱、各种花式作死、各种劈腿、各种约炮、各种堕胎，最后主角说：“我们的青春最终输给了现实。”―一段青春期堕落史。

 

　　战斗民族的青春期是这样的：【普京保镖抽空参加奥运会 夺得了拳击金牌】在里约奥运会上摘得金牌的俄罗斯拳击手叶夫根尼・季先科是总统的保卫人员之一。虽然季先科今年5月才轮岗至俄联邦保卫局，奥运会前的所有时间都在训练。俄联邦保卫局媒体和舆论联系中心评论称，“季先科确实是我们的人，代表俄联邦保卫局体育俱乐部。奥运会后他将回到体制内，即俄联邦保卫局。轮岗结束后做什么――是否留在我们这里――由他决定”。

 

　　小巴激动地说：这很俄罗斯！战斗民族嘛，抽空参加活动而已，一不小心就拿了个金牌！

　　公公深深地吸了一口烟：抽空？别闹了，普京的保镖平时的训练可比运动员专业多了。。。指不定哪下就给你放杀招了！

　　婷婷就想知道普京去的话，能拿下多少金牌？

 

　　【120岁印度僧人谈长寿秘诀:不近女色 每天练瑜伽】他目前申请吉尼斯世界纪录，确认他的年龄。吉尼斯世界纪录目前记载世上最长寿的男子，是日本人116岁又54天的木村次郎右卫门。席瓦南达身高158公分，来自圣城瓦拉纳西（Varanasi），成长环境极端贫穷，选择当僧侣。

 

　　　　他说，自己的长寿秘诀在“瑜伽、纪律和独身”。

 

　　老沙表示不服！虽然有老婆，但是这六字的长寿秘诀也掌握得很好啊！“瑜伽、纪律和独身”，经常各种姿势跪键盘堪比瑜伽，严格的家规家纪，独身从来不会和人民币出双入对！

 

　　吃饭的时候，老婆往老公碗里夹了块肉，温柔的说：你天天这么忙，回头给你找个经纪人吧？！老公顿时感动的热泪盈眶，放下饭碗，抚摸着她脸上被岁月磨砺的皱纹，深情的说：嗯，谢谢你为了这个家多年的付出，结婚到现在一次也没陪你出去走走，明天陪你逛逛八达岭野生动物园？

　　老虎们表示，带老婆来的太多，需摇号排队！

 

　　这年头，不一定非要到动物园才能见着老虎！【司机说商务车里装的是“波斯猫” 民警一查却是三只小白虎】一辆商务车绕开检查区域，朝后方驶入了停车场。商务车的举动引起现场民警的怀疑，立即上前对其进行检查。刚靠近车辆，民警就闻到一股臭味。他顺手打开后备厢，发现车里放了几个笼子，而里面装的动物样子有点奇怪。“乍一看，三只小家伙有点像幼年的老虎，可是端详许久后又觉得像某个品种的猫。”他拿不定主意，叫来同事一起看，他们初步确定这是白虎。

 

　　据说这个时候杭州方圆100公里内，苍蝇蚊子都要脱了裤子检查……这车主还想拿猫掩护老虎，这是犯法的呀！假装不懂是不管用的！
