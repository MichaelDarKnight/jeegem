民政部：7月灾情致804人死亡失踪-搜狐新闻

作者：管理员 发布于：2016-08-11 13:55:09
图片来源：视觉中国

　　近日，民政部、国家减灾委办公室会同国土资源部、住房城乡建设部、交通运输部、水利部、农业部、卫生计生委、统计局、林业局、地震局、气象局、保监会、海洋局、中央军委政治工作部、中国红十字会总会、中国铁路总公司等部门对2016年7月份全国自然灾害情况进行了会商分析。经核定，2016年7月份，我国自然灾害以洪涝、台风和地质灾害为主，风雹、旱灾、地震等灾害也均有不同程度发生。各类自然灾害共造成全国7018万人次受灾，612人死亡，192人失踪，454.1万人次紧急转移安置，192万人次需紧急生活救助；33.7万间房屋倒塌，153.6万间不同程度损坏；农作物受灾面积7290.8千公顷，其中绝收1515.5千公顷；直接经济损失2331亿元。综合来看，7月份全国灾情总体偏重，与2011－2015年同期均值相比，因灾死亡失踪人口、紧急转移安置人口、倒塌房屋和直接经济损失均偏多1倍以上。

　　2016年7月份，全国自然灾害主要呈现以下特点：

　　一是洪涝灾害南北并发、多地齐发。7月份，全国共遭受4次较大范围强降雨过程，其中，7月初长江中下游干流地区和18－21日华北地区相继出现今年入汛以来本地区最强降雨过程。长江出现2次编号洪水，海河南系漳河、子牙河发生1996年以来最大洪水，太湖发生流域性洪水，淮河辽河发生超警洪水。全国多地发生重大洪涝和地质灾害，上旬安徽湖北湖南贵州、下旬河北河南等地相继遭受重大洪涝灾害，给灾区人民生命财产和经济社会发展造成严重影响。据统计，洪涝灾害（含地质灾害）造成29个省（自治区、直辖市）1200余县（市、区）6120.9万人次受灾，671人因灾死亡失踪，414.4万人次紧急转移安置；31.6万间房屋倒塌，144万间不同程度损坏；农作物受灾面积5189.8千公顷，其中绝收1222.1千公顷；直接经济损失2132.4亿元。总的看，洪涝和地质灾害较2011年以来同期明显偏重，人员受灾情况、农作物受灾情况、倒损房屋数量和直接经济损失均为近年同期最高值，河北、湖北、安徽、河南等省灾情较重。

　　二是台风生成晚、致灾重。7月份共有2个台风登陆我国大陆地区，5个省（自治区）80个县（市、区）受灾。其中，第1号台风“尼伯特”于7月3日生成，生成时间偏晚，是强厄尔尼诺事件后正面登陆我国的首个台风，也是1949年以来登陆我国的最强初台。受其影响，闽江支流梅溪发生历史实测最大洪水，多条支流相继发生超警戒水位洪水，造成105人死亡失踪，“尼伯特”成为近5年来登陆我国造成因灾死亡失踪人数最多的台风。此外，7月下旬第3号台风“银河”影响广西、海南、云南等地，造成24万人受灾，直接经济损失逾3亿元。

　　三是 风雹灾害影响范围小 。 7月份，全国大风、冰雹等强对流天气造成23个省（自治区、直辖市）近450个县（市、区）遭受风雹灾害，受灾县数占全国总受灾县总数的2成，较2011以来同期均值偏少2成以上。据统计，风雹灾害造成326.7万人次受灾，18人因灾死亡失踪，5.2万人次紧急转移安置；4000间房屋倒塌，7万间不同程度损坏；农作物受灾面积466.2千公顷，其中绝收75.5千公顷；直接经济损失47.9亿元。其中，河北、内蒙古、湖北和云南4省（自治区）损失相对严重。总的看，7月份风雹灾情较2011年以来同期偏轻，死亡失踪人口为最低值，直接经济损失为次低值。

　　四是地震活动水平弱，旱灾损失轻。7月份，我国大陆地区发生5级以上地震1次，即7月31日广西苍梧县5.4级地震，发生次数为2011年以来同期偏低水平，此次地震造成近3000间房屋损坏，直接经济损失近5000万元。总的看，地震灾情较2011年以来同期相比明显偏轻。7月初，东北、西北部分地区相继遭受夏伏旱，因中下旬出现强降雨过程，旱情缓解。总的看，旱灾造成的损失较近年同期相比明显偏轻，农作物受灾、绝收面积为2011年以来同期次低值（仅高于2012年）。

　　五是城市内涝突出，对市民生产生活造成较大影响。 7月份，受极端强降雨和台风等影响，一些大城市和中小城镇频繁遭受内涝，武汉、合肥、石家庄、太原等多个城市严重积水，对市民正常出行和生产生活造成较大影响。其中，6月30日－7月6日，湖北武汉市降雨量突破该市自有气象记录以来周持续性降水量最大值，城区内涝严重；18－21日河北石家庄、河南安阳等北方城市降雨量超历史极值，城市积水严重。

　　六是倒损房屋数量多，受灾群众转移安置任务重。 7月份，各类自然灾害造成全国33.7万间房屋倒塌，153.6万间不同程度损坏，为2011年以来同期最高值，较同期均值增长9成以上，受房屋大量倒损影响，7月份紧急转移安置人口急剧增加，为2011年以来同期最高值，较历史同期均值增长1倍以上，灾害救助和临时安置任务十分繁重。
