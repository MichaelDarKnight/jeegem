网传特警暴打超载冲卡司机 警方称已非常隐忍-搜狐新闻

作者：管理员 发布于：2016-08-22 16:23:50

　　河南省襄城县交通运输执法局被曝在前天（8月17日）凌晨查处超载货车司机时，执法人员联合民警殴打司机常某及司机父亲。当地一居民称，事发于凌晨两三点，常某因超载被拦后开车逃逸，后在家附近被追上，民警将司机拉下车后殴打，从家中赶来的司机父亲也遭殴打，伤势严重。今天上午，警方发表通稿称，常某蓄意闯卡，逃离过程中故意冲撞警车。常某到家附近后停车后将车反锁，民警劝说无果后破窗带走常某，常某亲戚和特警队员发生肢体冲突，并谩骂殴打警员。目前，常某及其父亲、三名特警受伤。

　　当地居民：司机超载被查后狂奔 特警追上后拖出暴打

　　网传视频显示，现场有一辆货车、两辆特警车以及一辆交通执法车，数百名群众聚集在车辆旁边，现场还有多名特警和民警。一段疑似司机受伤后的视频显示，一名裸着上身的男子躺在地上，有人用灯光照射脸上，但该男子没有反应，仍然紧闭双眼，表情痛苦。

　　据大河论坛网友介绍，8月17日凌晨，一辆大车途经311国道襄县库庄附近时遭到由交通综合执法、交警和特警组成的联合执法队的追赶。在后有追兵的情况下，大车司机驾车行驶到自己家门口，结果一帮人上前将大车玻璃砸碎，将司机拉下来一顿暴打，至使司机和他父亲受伤，司机父亲伤势尤为严重，在政府的安排下已经从当地医院转入省知名医院进行救治。他还称，据知情人说，司机父亲肋骨被打断，胸腔积血，气管被切开。

　　一名当地居民补充说，当时车上有司机、司机妻子和孩子三人，货车被追上后司机被特警拉下车暴打，从家中赶到的货车父亲也遭到特警殴打。司机和他父亲都受了重伤，“听说司机父亲进了重症监护室”。随后，现场的两辆特警车和一辆交通执法车被群众围住不让走。直到当天下午，打人者在民警护送下离开，群众才散开。

　　不过，也有网友对上述说法持不同看法。大河论坛网友“不是很懂”称，司机“关大灯、强行闯卡，别人家特警的车”。网友“那年花开半夏”则说，司机超载被查，硬闯两个关卡，然后被拦截。他还称，“撞警车，玻璃碎，车胎爆了，车都差不多毁了”。

　　此外，当地一名货车司机介绍，当地曾多次发生过大车被执法人员追赶的事情，“很容易发生事故”。而襄城县交通运输执法网也报道称，8月12日凌晨，襄城县交通运输执法局执法人员在执勤时，发现一辆大型拉货车涉嫌违法超限运输，执法人员立即示意停车检查，但是，该车辆驾驶人无视执法人员指挥，驾驶车辆疯狂冲卡逃窜，在逃窜过程中将一辆交通执法车辆撞击受损，最终，在交警、公安的配合下将该车辆暂扣。

　　警方回应：司机闯卡撞警车后起冲突 民警司机均受伤

　　昨天中午，库庄镇政府工作人员回应网传货车司机被特警打一事称，共有两人受伤，镇政府也曾前往慰问，但没有透露详情。

　　今天，当地公安局发表了一篇通稿《襄城县公安局妥善处理一起货车司机闯卡阻挠执法事件》，详细介绍了警方掌握的情况。

　　襄城县公安局法制室主任张中灿说，基本确定常某驾驶大货车蓄意闯卡，并在逃离过程中故意冲撞警车，其行为均已涉嫌阻碍执行公务犯罪，其家属故意阻挠执法，谩骂殴打警员的行为已涉嫌妨碍执行公务。

　　该县交警大队城区中队中队长孙晓峰介绍说，当时交通运输执法人员向一辆大货车喊话示意其靠边停车接受检查时，司机向后倒车后向右打方向加速闯卡。“当时我看到这一情况，高声向货车司机喊话让他停车接受检查。”孙晓峰说，货车贴着路沿石，几乎是擦着执法车辆挤过去后，加速逃离。

　　襄城县公安局特警大队副大队长刘旭兵称，特警车被撞开后，货车行驶到库庄镇卫生院门口附近停下。执勤民警要求常某下车接受检查无果，随后破窗打开车门，将常某“带上警车”，但常某父亲在旁阻拦，与特警队员发生肢体冲突，阻碍执法大队执法。此时，有多名和事件无关人员聚集在停车地附近起哄，说警察打人，“后经了解这些起哄的都是长期经营超载大货车的车主和其亲属”，“起哄就是为了煽动群众”。

　　公安局局长：警员面对谩骂和殴打，表现了克制和隐忍

　　接警后赶到现场的襄城县公安局库庄派出所民警李伟兵则说，他到现场后，常某和他父亲自称受伤，后被送上救护车。救护车走后，常某三婶骂骂咧咧追打多名特警队员，多次冲到队员身边朝脸上扇，特警队员没有还手。常某四叔还将特警车辆的四个轮胎气放掉。

　　襄城县公安局局长孙刚杰表示在处理这起大货车恶意闯卡事件中，特警队员对交通运输执法局的配合积极到位，“面对嫌疑人家属侮辱谩骂和殴打，表现了克制和隐忍。”

　　针对嫌疑人提出的，在被带离过程中，有警员对其进行殴打，已成立由督察、纪检、法制、刑侦等部门组成专门工作组，根据伤者伤情，详细调查经过，一经查实，将依法依规对相关人员给予处理。

　　孙刚杰说，“我们注意到，事件发生后，有个别人员在网络上和微信朋友圈发布、散播视频和文字，捏造公安打死人、公安抢小孩等谣言，目前网警已严密监控，对恶意制造谣言、传播谣言者，将会同有关部门依法予以严厉打击。”

　　通稿称，目前货车司机常某和其妻子、父亲，正在襄城县人民医院接受检查，法医张俊涛给出了鉴定结果，“司机常某、他的父亲及我局的三名特警队员均有不同程度的擦伤”。不过，并未提及常某受伤原因。
