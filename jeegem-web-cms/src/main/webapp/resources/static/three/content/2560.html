三星新1代TabPro S曝光：搭完整版Win10-搜狐数码

作者：管理员 发布于：2016-09-01 17:48:11

据外媒Sammobile报道，三星二合一平板Galaxy Tab Pro S上市之后，一直被誉为微软Surface Pro的请有力竞争者。现在，消息人士头，三星新一代Galaxy Tab Pro S已经开始研发，并计划在未来几个月内发布。



三星新1代TabPro S曝光

    报道称，三星Galaxy Tab Pro S2平板将延续Galaxy Tab Pro S的设计风格，同样采用金属机身设计和分辨率为2160×1440像素的12英寸Super AMOLED显示屏。仍然搭载Windows10系统将有SM-W728、SM-W727、SM-W723和SM-W720等四个型号。

     Galaxy Tab Pro S2将有望搭载最新的Intel Core M系列处理器。据了解，英特尔在日前已经发布了第七代酷睿（Core）处理器Kaby Lake。按照英特尔的计划，Kaby Lake构架下的处理器在每瓦性能上是第六代的8倍；网络性能比之前高出19%；生产力应用程序性能则要高出12%。从今年9月份开始，配备Kaby Lake处理器的电脑将陆续出货。

     尽管现有消息显示三星Galaxy Tab Pro S2平板将无缘即将开幕的IFA2016展会，但是消息人士预计，三星新一代Tab Pro S可能会跟前作一样在2017年初的CES展会上才会亮相。
