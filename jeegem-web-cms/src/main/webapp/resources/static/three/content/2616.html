女儿在宿舍受气 父母来校“帮忙”殴打4名舍友-搜狐新闻

作者：管理员 发布于：2016-09-16 23:45:09

　　华商报商洛讯（记者 陈永辉）女儿与舍友发生矛盾，家长到学校帮娃“出头”，殴打了4名女生；被打女生家长闻讯赶来，双方发生冲突，一名家长被打骨折住院。这一幕就发生在商洛职业技术学院。

　　被打学生：

　　舍友家长拽着我头发往墙上撞

　　9月14日上午，华商报记者在商洛市中心医院见到了被打的4名女生，她们正在拍片检查，4人均不同程度出现头痛症状。“我当时刚准备去上课，小郭的父母问我叫啥，我就说了名字。”被打学生小胡说，没等她反应过来，对方直接过来拽着她的头发就开始踢她。

　　“我当时在隔壁宿舍，有同学叫我，我们去挡架也被打了。”小陈说，小郭的家长扇了她一巴掌，还拽着她的头发往墙上撞，现在她还感觉头疼得厉害。小王称，小郭早上起来后出去了，过了一会儿就和她父母一块上来，先打的是小胡，后打的是小崔，扯着头发往墙上碰，还拿刷牙杯子往头上砸。

　　几名女生说，前一天是小郭扫宿舍，她没有扫地，她们就说不扫地第二天接着扫，小郭并未理会。“我们一个宿舍的，大家也很少和她说。可能是因为打扫卫生的事情，她给父母说了。”

　　一位家长拿着孩子一小撮头发气愤地说，这就是被小郭家长拽下来的。不管怎么说，家长跑到宿舍不合适。有知情家长称，“小郭的父亲是某卫生院的院长，另一方孩子的家长也是公职人员，发生这样的事，实在是不应该”。

　　家长否认打过孩子舍友

　　“发生冲突的学生是五年制护理专业3班一个宿舍的。”商洛职业技术学院学生保卫处于姓相关负责人表示，事后他们了解到，早上7点半左右，学生小郭的家长来到女生宿舍楼，楼管还挡了没拦住，他们直接跑到5楼宿舍，学生就给班主任打电话。结果班主任刚走到宿舍门口，一位被打学生的家长已闻讯赶到，双方家长发生了冲突。该负责人称，学校已开始调查，随后会做出处理。

　　之后记者在商洛市第三人民医院见到了受伤的小郭父母，郭先生肋骨骨折，而他的妻子腰部等处受伤。郭先生说，13日晚孩子给他打电话，称在学校被同学欺负了，“我之前就给学校说让调换宿舍，娃的舍友知道了，就针对娃，听娃说被欺负了我就特别生气。我并没有打她们，只是说了几句。”郭先生表示，早上他们从宿舍楼出来后，女儿一个舍友的家长就守在宿舍楼下，看到他们就上去殴打，将他打倒在地，他的妻子上前阻挡也被打伤。

　　郭先生的妻子否认去宿舍打女儿舍友，还说自己的脸也被女儿舍友抓伤了。郭先生说，自己是公职人员，出了这样的事情，他也不想让事情闹大，不想对孩子有影响。小郭给记者提供的照片显示，当时郭先生躺在地上，一名男子欲上前打人，其他几人忙上前阻挡。“昨天晚上在宿舍，4名舍友踢了我，我就给父母说了。”小郭说，舍友打她可能和自己要换宿舍等事情有关，之前她与舍友之间一直就有矛盾。

　　目前，商州警方已介入调查。
