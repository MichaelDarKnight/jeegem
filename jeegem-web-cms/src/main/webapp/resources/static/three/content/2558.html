交行一支行行长自杀 称给国家财产带来巨大风险-搜狐新闻

作者：管理员 发布于：2016-09-01 17:48:11

　　段云生前工作的交通银行科大支行。

　　9月1日上午，澎湃新闻（www.thepaper.cn）从权威渠道获悉，中国交通银行一名支行行长段云在湖南省长沙市岳麓区骑龙大街中心广场自杀身亡。警方在死者生前驾驶的车辆中发现一封遗书。遗书中称，因为某公司贷款一事被骗了，“给国家财产带来了巨大风险，而深深自责”。

　　权威渠道信息称，8月28日22时许，岳麓区公安机关接到群众报警，称段云（男，44岁，户籍地伟开福区科大家园北苑，系中国交通银行科大支行行长）于8月27日上午出门后与家人失联，家属在岳麓区天顶街道骑龙大街前坪发现了其驾驶的小汽车，但未找到本人，请求公安机关查找下落。接警后，岳麓区公安机关对骑龙大街进行了地毯式搜查。8月29日凌晨2时许，民警在骑龙大街中心广场五楼发现了悬吊于一门店北侧龙骨架上，已无生命体征的段云。同时，民警在段云生前驾驶的车辆中发现了一封遗书，因为某公司贷款一事被骗了，“给国家财产带来了巨大风险，而深深自责”。

　　综合前期现场勘查、尸体勘验、视频侦查及遗书分析，公安机关初步判断段云为自杀。

　　截止8月30日18时许，段云的遗体仍停放在殡仪馆，遗体及笔迹鉴定仍在进行中。

　　9月1日，澎湃新闻来到交通银行科大支行询问段云生前工作情况及死亡信息，一名男性保安称段云已不在银行上班，并怒斥记者不要多问，不要乱写。截止记者发稿，中国交通银行尚未对外公开段云的死讯，也未对其遗书进行公开说明。
