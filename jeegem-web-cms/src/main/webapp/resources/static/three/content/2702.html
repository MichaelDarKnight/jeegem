战斗民族的客机！冲开特浓大雾降落首都机场-搜狐新闻

不是所有雾霾都叫特浓雾霾

　　这次恶劣天气过程中雾霾没抢上头条

　　占据头条位置的竟然是一架俄罗斯航空飞机

　　事情得从11月4日那个寂静岭一般的夜晚讲起，北京已经发布了雾霾橙色预警，气象部门也发布了特大浓雾预警。

　　受天气影响，首都机场降落条件不行，空管部门联合机场启动了二类盲降（就是不用肉眼，凭借仪表等辅助降落）。

　　然鹅……近地面的雾霾啊，就像一口浓痰，吹都吹不散。飞机还是不能正常降落。

　　据空管局的小鲜肉说，工作这么长时间了，11月4日晚上8点到10点那两个小时，经历了一年的

　　“7700”

　　

　　知道君解释一下“7700”：表示遇险。现代民航飞机一般装有应答机，可以提供四位数字编码供地面雷达识别。一般地面空管人员根据雷达显示的应答机编码识别天上的飞机谁是谁。而如果飞机在空中遇险（如燃料不够，起火，发动机停车等等），飞行员会将应答机编码调为7700，地面空管人员则会立即进行应急处置。

　　据说，当天首都机场上空是这样的。

　　一架从香港赤�v角机场飞往首都机场的航班，就呼哧带喘的奋力拼搏在这迷之上空，三次试图降落在首都机场都未成功，最后返航香港。

　　网友评论也是神级，壶口瀑布也出来了。

　　据知道君从内部打听到，香港这架飞机的机长还算是艺高人胆大的，比较熟悉北京的雾霾天降落，他当天降落在视程高度的时候发现眼前的情景心想：哟，今儿这个雾霾有点浓啊，先落落试下吧。

　　没成想，一猛子扎进平流雾里发现根本看不见跑道灯，直接拉起操纵杆复飞。

　　地面这大片浓（痰?）浓（雾?）被腾空的飞机划开了一条道。机长一瞅，复飞操作能带起地面浓雾，说不定再来一次就能降了，他急忙又盘旋摆正降落姿势。

　　第一次被划开的浓雾没多久又密密得聚集在跑道上，这次降落又没成功，往复三次未果，机长这哥们表示不来了，油够咱还是回香港吧。

　　知道君查了下，11月4日晚上，首都机场是红色延误预警，空管提供的数据则显示，首都机场本场的能见度一度降低到200米一下，虽然启动盲降也不能满足航班起降，所以，当天就出现辣么多降不下来的飞机备降到外地。

　　不过，就在这个时候，“战斗民族”开的飞机安全降落在首都机场，人家不但降落了，还匪夷所思的提前到达。

　　那些对“战斗民族”早有耳闻的吃瓜群众，呼拉拉的起立鼓掌。

　　网友@_fengsuiyingdong_表示：论机长，我只服战斗民族。更有网友直呼胸臆：战斗民族不需要解释。

　　前排坐下，且听知道君跟你们扯一篇“战斗民族”为啥开飞机这么猛。“战斗民族”为什么这么胸残！！！！

　　为什么俄罗斯客机能被允许在台风、冻雨等极端恶劣的天气条件起降?实际上,每家航空公司都可以在不逾越民航主管部门规定底线的前提下,根据各自机队情况、保障情况和飞行员水平制订相应的飞行标准。可以看出,俄罗斯的航空公司制订的飞行标准对天气条件的要求明显更低。

　　毛子哥生存环境是啥样的大家想想,人家多数地方,一年有大半的时间被冰雪覆盖,频繁的暴风雪天气令飞行条件非常恶劣。面对这样的条件,俄罗斯的航空公司、机场只能主动适应。所以,俄罗斯民航的飞行员绝对要首先具备在极其恶劣条件下起降的本领。试想一下,如果下一点小雪就导致机场大面积延误,俄罗斯则可能有大半年的时间不适合坐飞机旅行,这样的话,俄罗斯的民航业可以说根本没有发展前途。

　　另外，俄罗斯民航也是从军航脱出来体制，这就要说“战斗民族”的优势了，军航老人带徒弟，一代一代的彪悍传统肯定还沿袭着，飞机开得像战斗机，也就不难理解了。

　　不过，俄罗斯航空在历史上也曾因为太猛太随意造成了悲惨的空难。

　　1986年苏联一个机长跟副驾驶打了个赌说自己可以把驾驶舱窗帘全部拉上不看外面，只靠纯仪表盲降让这架TU-134客机降落。结果最后降落时他不顾飞机的近地警告，着陆速度过大，坠毁在跑道上，造成机上94名乘客和机组成员中的70人死亡，副驾驶也一同遇难。

　　从这一点来看，知道君必须为咱们中国的民航安全性要求点个赞。

　　中国民航整个系统，根深蒂固的一个生产底线就是“安全”，俗话说，为了安全，什么都可以不要。不久前，浦东机场的跑道入侵事件暴露出了安全问题，民航局就严肃查处了相关责任人，为了保证飞机之间的安全距离，浦东机场宁愿舍弃了一部分准点率，也要保证安全。

　　这不是知道君自己猜度，有数据证明。进入21世纪后，中国民航的安全记录确实很好，上一次发生致死的空难是在2010年8月24日，河南航空VD 8387伊春空难。而上上次，则要追溯到2004年11月21日，东方航空MU 5210航班在包头发生空难。

　　知道君坚信一条家训：出门在外，安全第一。
